13. Scope delle Variabili: Statico e Dinamico

Lo scope (ambito di visibilità) di una variabile definisce dove essa può essere usata all'interno del programma. Esistono due principali modalità di determinazione dello scope:

  • Scope statico (lessicale): lo scope è determinato in fase di compilazione, in base alla struttura del codice. Le variabili sono accessibili solo all’interno del blocco in cui sono state dichiarate.
  • Scope dinamico: lo scope viene risolto in fase di esecuzione, seguendo la catena di chiamate alle funzioni. Le variabili vengono cercate nei contesti delle funzioni chiamanti.

Di seguito alcuni esempi scritti in linguaggio C che mostrano il comportamento delle variabili nei due tipi di scope.

Ambito o Scope Statico

void f() {
  int x;
  ...
  x = 5; // OK
  ...
}

void g() {
  ...
  x = x + 1; // KO (x non visibile qui)
  ...
}

void h() {
  int x;
  ...
  x = 4; // OK
  ...
}

Ambito o Scope Dinamico – Esecuzione 1

void f() {
  int x;
  x = 5;
}

void g() {
  x = x + 1; // usa x di f()
}

void h() {
  int x;
  x = 4;
}

Chiamate: f(); g(); h();
Risultato: x = 6

Ambito o Scope Dinamico – Esecuzione 2

void f() {
  int x;
  ...
  x = 5;
  ...
}

void g() {
  ...
  x = x + 1;
  ...
}

void h() {
  int x;
  ...
  x = 4;
  ...
}

Chiamate: f(); g(); h();

Ambito o Scope Dinamico – Esecuzione 3

void f() {
  int x;
  x = 5;
}

void g() {
  x = x + 1;
}

void h() {
  int x;
  x = 4;
}

Chiamate: f(); h(); g();
Risultato: x = 5
