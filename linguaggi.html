13. Scope Statico vs Scope Dinamico

Ambito o Scope Statico: le variabili sono accessibili solo nel blocco in cui sono dichiarate. Le funzioni non condividono le variabili a meno che non siano dichiarate in uno scope comune.

void f() {
  int x;
  ...
  x = 5; // OK
  ...
}

void g() {
  ...
  x = x + 1; // KO: x non è visibile qui
  ...
}

void h() {
  int x;
  ...
  x = 4; // OK
  ...
}

Ambito o Scope Dinamico: le variabili sono risolte in base all’ordine delle chiamate a runtime. La funzione g() può accedere a x se chiamata da un contesto che lo contiene.

Esecuzione 1

void f() {
  int x;
  x = 5;
}

void g() {
  x = x + 1;
}

void h() {
  int x;
  x = 4;
}

Chiamate: f(); g(); h();
Risultato: x = 6 (x di f() viene aggiornato da g())

Esecuzione 2

void f() {
  int x;
  ...
  x = 5;
  ...
}

void g() {
  ...
  x = x + 1;
  ...
}

void h() {
  int x;
  ...
  x = 4;
  ...
}

Chiamate: f(); g(); h();
// Dipende dal contesto di chiamata dinamico

Esecuzione 3

void f() {
  int x;
  x = 5;
}

void g() {
  x = x + 1;
}

void h() {
  int x;
  x = 4;
}

Chiamate: f(); h(); g();
Risultato: x = 5 (x aggiornato nell’ambito corrente dinamico)
